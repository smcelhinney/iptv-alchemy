Main features.

Xtream Codes support

Connect to any Xtream Codes provider to download M3U/XMLTV feeds, extract categories, and generate filtered output ready for indexing and playback.

TMDB integration

Enrich movies and series with posters, synopses, cast, and ratings from The Movie Database for a polished, metadata-rich library experience.

Subtitle support with OpenSubtitles

Search and download subtitles directly from the player UI, then adjust offset and size on the fly with client-side VTT rewriting and CSS ::cue styling.

Self-hosted for security

Run the full Docker Compose stack — Meilisearch, Redis, Flask API, and React UI — on your own infrastructure so your viewing data stays private.

Media Server support

Generate .strm files and filtered M3U/XML output for Emby and Jellyfin, turning your IPTV sources into a managed media library.
